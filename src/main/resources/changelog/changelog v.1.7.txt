AranarthCore v. 1.7

Features
[+] Added /ac aranarth to provide a book filled with information about Aranarth
[+] Added /ac date to display the current server date
[+] Added /ac dateset to change the current server date in-game
[+] Added particle effect around the bottom of waterfalls
[+] Can re-add bark to stripped wood and create 6-sided wood from logs using bone meal
[~] Coal is now crafted using 2 charcoal and not just 1
[~] Prevented shops from being created in non-survival worlds
[+] Added sounds for when players join and quit the server
[+] Added support for /ac potion remove <qty> to remove potions from the player's potion inventory
[~] Prevented the 111th of each month from showing as 111st in date message
[~] Prevented wandering trader death throwing error when slain by i.e fall damage or non-mob source
[~] Prevented issues where /ac commands couldn't be run by console
[~] Added missing response messages from commands executed by Console
[~] Fixed bug where enhanced aranarthium ingots could not be crafted
[~] Snow will no longer fall in jungle biomes, only mountainous parts of it
[~] Increased snow radius from 100 blocks to 250
[+] Players can store their EXP in an EXP bottle by sneak right-clicking
[~] Fixed issues with rain not starting/stopping correctly
[~] Added feature to consider storms from previous server restart
[+] Rain messages display in all months
[~] Restructured how rain and thunder are started and managed
[~] Fixed issues where rain and thunder was not starting and ending correctly

Miscellaneous
- Restructured date message logic to be a callable method
- Minor adjustments to the date message to remove extra lines and AranarthCore prefix to all messages
- Minor changes to wandering trader slain message
- Added helper method to quickly list the quantity of each potion a player has
